# Globelinkers

A travel discovery web app in the spirit of TripAdvisor — browse destinations, read reviews, and get personalized recommendations.

## Features
- Destination listings with details and photos
- Search & filter by location/category
- User-submitted reviews and ratings

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Status
🚧 Repository scaffold — core application code is being migrated in.

## Setup
```bash
npm install
npm run dev
```
